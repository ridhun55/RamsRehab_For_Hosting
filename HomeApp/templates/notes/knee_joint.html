Knee Joint

Therapy 1

ANTERIOR CRUCIATE LIGAMENT INJURY

- Torn ACL in the knee joint during twisting movements
- Pain during movement
- Swelling
- Atrophy of quadriceps muscle

Special Test

- Anterior drawers test
- Lachmanns test

Treatment

- Electrotherapy
- Ultrasound
- TENS
- Cryotherapy

Therapeutic Exercise

PHASE : 1
  -  Patellar mobilization
  -  Static quadriceps
  -  VMO strengthening using a ball
  -  Hamstring stretching in sitting

PHASE : 2
  -  Quadriceps strengthening using theraband
  -  VMO strengthening using theraband
  -  Single leg standing
  -  Squatting
  -  Aquatic activities

PHASE : 3
  -  Single leg squatting
  -  Plyometric deep jumb
  -  Ice skating

Therapy 2

MENISCAL TEAR

- Can be caused by a single injury or over time in people who are active and older people
- Recurrent locking of knee joint

Special Test

- Palpating joint line tenderness
- McMurray’s test
- Teslay’s test

Treatment

- Electrotherapy
- TENS
- Cryotherapy

Therapeutic Exercise

- Static quadriceps
- Static hamstring
- Passive knee flexion
- Resisted knee extension
- Single leg bridging
- VMO strengthening
- Clamshell exercise
- Kinesio taping

Therapy 3

ILIO-TIBIAL BAND SYNDROME

- Friction between IT band and knee leads to pain
- Pain is severe when there is tension in IT band
- Most tension in 20-30 degree knee flexion

Treatment

- Cryotherapy
- Dry needling
- TENS

Therapeutic Exercise

- Side lying abductor strengthening using Theraband
- Resisted side stop with Theraband
- Hydrants
- IT band stretch

Therapy 4

KNEE OSTEOARTHRITIS

- Joint pain
- Morning stiffness of knee joint
- Decreased muscle strength

Special Test

- Knee PROM with overpressure

Treatment

- Electrotherapy
- TENS
- Ultrasound
- Cryotherapy
- Dry needling

Therapeutic Exercise

- Static quadriceps
- Static gluteus
- VMO strengthening
- Clamshell exercise
- Quadriceps strengthening in sitting
- Static bicycle

Manual therapy

- Knee flexion mobilization

Therapy 5

HAMSTRING STRAIN

- Common in high speed sports
- Occures when hamstring stretches when it is about to contract

Special Test

- Manual muscle testing of hamstring

Treatment

- Electrotherapy
- TENS
- Ultrasound
- Cryotherapy

Therapeutic Exercise

- Static quadriceps
- Static hamstring
- Bridging
- Swiss ball hamstring curls
- Prone plank
- Single leg stance
- Kinesiotaping
